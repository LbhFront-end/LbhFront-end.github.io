Prettier

官网

什么叫Prettier？

Prettier是一个固定的代码格式化程序，支持：

  • JavaScript，包括ES2017
  • JSX
  • Angular
  • Vue
  • Flow
  • TS
  • CSS，Less和SCSS
  • HTML
  • JSON
  • GraphQL
  • Markdown，包括GFM和MDX
  • YAML
